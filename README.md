## description

`code-rescue` is a senior engineer you install when your codebase is on fire. Specializes in **legacy migrations**, **production incidents**, and the kind of bugs that show up at 2am. No retainers, no six-week onboarding — run `npx code-rescue` today, ship tomorrow.
